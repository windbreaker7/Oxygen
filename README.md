## ✨ Key Features

### 🎯 Precision Aimtouch
* **Center-Locked FOV:** A static "Kill Zone" locked to your screen center.
* **Dynamic Smoothness:** Fully adjustable lerp-based smoothing (1% - 100%).
* **Wall Check:** Intelligently ignores targets behind solid objects.

### 👁️ Advanced Visuals (ESP)
* **Player ESP:** Track real players with low-latency BillboardGuis.
* **Scav & Loot ESP:** Highlights AI enemies and valuable world items.

### 🛡️ Stealth & Security
* **Shadow Bypass:** Metatable hooks to nullify `ExecutorDetection` remotes.
* **Integrity Scanner:** Built-in tool to check if anti-cheat signatures have changed.

### 📜 Live Changelogs
* **GitHub Sync:** Fetches patch notes directly from this repository.
